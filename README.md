# AI / Machine Learning Foundations – Assignment II

## Bike Sharing Demand Prediction

This repository contains my Assignment II submission for the AI / Machine Learning Foundations course.

### Objective
The goal of this project is to predict hourly bike rental demand using the UCI Bike Sharing Dataset (`hour.csv`) and compare the performance of multiple regression models:
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

The project includes:
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Cyclical Encoding
- Model Training
- Hyperparameter Tuning
- Residual Analysis
- Bias–Variance Evaluation
- Final Model Selection

## Files
- `assignment_2_MOHAMED_EMBABY.ipynb` → Main notebook
- `hour.csv` → Dataset used for training and evaluation
